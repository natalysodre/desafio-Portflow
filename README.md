# Projeto PortFlow :sparkles:
## Módulo 1 do curso de Desenvolvedora Front-End do Vai Na Web :computer:



Neste projeto, desenvolvi uma página web que simula um site em portifólio. 

O objetivo do desafio era praticar flex-box, bem como fidelidade ao design proposto e otimização para desktop/dispositivos móveis. 



A página, portanto, foi desenvolvida a partir de tecnologias web padrão, como HTML e CSS.
